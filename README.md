# big-market-go

## Overview

This is a lottery module implemented using go-zero.

## environment
- Mysql:8.0
- Redis
- go1.22

## Quick Start
```shell
git clone https://github.com/HSn0918/big-market-go.git
cd big-market-go
go mod tidy
cd app/strategy/raffle/api
go run strategy.go
```
